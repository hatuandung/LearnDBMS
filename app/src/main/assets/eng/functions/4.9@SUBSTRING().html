SQL SUBSTRING() Function

The SUBSTRING() function

The SUBSTRING() function is used to get part of a string.

SQL SUBSTRING Syntax

This function is called differently for the different databases.

MySQL Syntax

MySQL supports two ways of using this function:

SUBSTR(column_name, start_position[, length])
FROM table_name

or

SUBSTRING(column_name, start_position[, length])
FROM table_name

Oracle Syntax

SUBSTR(column_name, start_position[, length])
FROM table_name

SQL Server Syntax

SUBSTRING(column_name, start_position, length)
FROM table_name

Parameter Description
column_name Required. The field to extract characters from
start Required. Specifies the starting position (starts at 1)
length Optional. The number of characters to return. If omitted, the SUBSTRING() function returns the rest of the text

Note: following syntax is not supported in SQL Server:

SUBSTR(column_name, start_position)

SQL SUBSTRING() Examples

The "Persons" table:

P_Id LastName FirstName Address City
1 Jameson John Streets 15 Sander
2 Smith Kate Green 68 Sander
3 Kristensen Olya Ski 2 Stavn

Now we want to get the first characters from "FirstName" column.

We use the following SELECT statement:

SELECT LastName, SUBSTR(FirstName, 1, 1) AS Initial
FROM Persons

The result-set will look like this:

LastName Initial
Jameson J
Smith K
Kristensen O

In the second example we want to get the last two characters from the "FirstName" column.

We use the following SELECT statement:

SELECT LastName, SUBSTR(FirstName, -2, 2) AS LastChars
FROM Persons

The result-set will look like this:

LastName LastChars
Jameson hn
Smith te
Kristensen ya
