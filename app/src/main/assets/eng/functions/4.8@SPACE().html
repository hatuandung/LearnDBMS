SQL SPACE() Function

The SPACE() Function

SQL SPACE() function is used to return space characters, the number of the characters is indicated by the interger parameter.

SQL SPACE() Syntax

SELECT column_name SPACE(number_of_spaces)
FROM table_name

Note: SPACE() function is supported by MySQL and MS SQL Server, but not Oracle.

SQL SPACE() Example

This time we will not use the table, just add 5 spaces between words.

We use the following SQL statement:

SELECT 'Hello' + SPACE(5) + 'World!'

The result-set will look like this:

Hello     World!
