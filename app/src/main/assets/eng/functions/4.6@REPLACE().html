SQL REPLACE() Function

The SQL REPLACE() function is used to replace every occurrence of a specified string with a replacement string.

SQL REPLACE() Syntax

SELECT REPLACE(column_name, string_to_replace, string_replacement)
FROM table_name

Note: If any of the string expression is NULL, the function returns NULL. If no match found, the function returns column_name strings without change.

SQL REPLACE() Example

We have the following "Orders" table:

O_Id OrderDate OrderPrice Customer
1 2011/11/07 1200 Johnson
2 2011/08/15 1500 Brown
3 2011/05/04 600 Johnson
4 2011/05/21 500 Johnson
5 2011/07/30 2500 Taylor
6 2011/10/08 300 Brown

Now we want to replace all records which matches "Brown" in "Customer" column with "unknown".

We use the following SQL statement:

SELECT O_Id, OrderDate, OrderPrice, REPLACE(Customer, 'Brown', 'unknown')
FROM Orders

The result-set will look like this:

O_Id OrderDate OrderPrice Customer
1 2011/11/07 1200 Johnson
2 2011/08/15 1500 unknown
3 2011/05/04 600 Johnson
4 2011/05/21 500 Johnson
5 2011/07/30 2500 Taylor
6 2011/10/08 300 unknown
